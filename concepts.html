ARPANET

ARPANET was the precursor to the web, created during the Cold War as a response to the nuclear threat of Soviet weaponry (Featherly, 2026). Like many human inventions it was made during war. Before it, data had to move via physical tapes, direct connections, or inputs.

Before After
Physical tapes Instant network transfer
Direct wired connections Any computer to any server
Manual inputs Real-time data sharing

How Web Servers Work

Concept 3

Web servers are the pool of information your computer accesses, similar to a library. If a request is sent and accepted, the server allows the client to access HTML files and app data. There are two types, both allow the user to access data.

Concept 4

History of Communication

Humans spent millennia trying to connect. Knowledge was shared orally, like a generation game of telephone. Then came writing in Mesopotamia. Then printing in 9th century China (History.com Editors, 2018). The web is the next step in that process.

Data Collection

Concept 5

Dynamic servers allow collection of massive amounts of data that can be filtered and used to track, monitor, and sell to users (Microsoft Azure, n.d.). Databases are one of the most integral components of many tech companies.

An internet without web servers would be the equivalent of the Library of Alexandria without scrolls.
